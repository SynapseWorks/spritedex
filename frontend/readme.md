# SpriteDex Frontend

The frontend provides:
- species browsing
- encounter logging
- maps
- scanning workflows
- collection systems
- future AR interfaces

---

# Planned Stack

- React
- Next.js
- Tailwind CSS
- TypeScript
- MapLibre or Leaflet

---

# Primary Views

## Home Dashboard
Overview and recent activity.

## Scanner View
Photo upload and AI suggestions.

## Species View
Species information pages.

## Encounter Journal
Personal encounter history.

## Map View
Geographic visualization of encounters.

## Collection View
Discovered species tracking.

---

# Design Goals

The frontend should feel:
- calm
- immersive
- exploratory
- scientific
- solarpunk
- field-ready
