# SpriteDex

SpriteDex is a real-world Pokédex for plants, animals, fungi, insects, and ecological encounters.

The goal is to create a personal ecological intelligence system that combines:

- species reference data
- AI-assisted identification
- GPS encounter logging
- field journal functionality
- maps and habitat tracking
- seasonal observations
- medicinal and ecological knowledge
- future AR and wearable integration

SpriteDex is designed as part of the broader Project Sprite / AetherOS ecosystem.

---

# Core Concept

SpriteDex has two primary layers:

## SpeciesDex
The canonical database of known species.

## Encounter Journal
The user's personal observations and experiences with those species.
