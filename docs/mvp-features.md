# MVP Features

The goal of the MVP is to create a functional ecological encounter tracker.

---

# MVP v0.1

## Species Database
- Store known species
- Scientific and common names
- Categories and taxonomy

## Encounter Logging
- Record sightings
- GPS locations
- Photos
- Notes

## AI Suggestions
- Species candidate suggestions
- Confidence scores

## Collection Tracking
- Track discovered species
- Encounter counts

## Map View
- View encounters geographically
