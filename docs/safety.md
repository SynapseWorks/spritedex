# Safety Philosophy

SpriteDex is an educational and observational tool.

AI identification suggestions are advisory only.

---

# Important Rules

- Never consume organisms based only on AI identification.
- Dangerous species require warning labels.
- Mushroom identification should be treated with extreme caution.
- Users should compare multiple sources before consuming wild organisms.

---

# Planned Safety Features

- confidence scoring
- lookalike comparisons
- warning banners
- toxicity labels
- verification indicators
- human confirmation recommendations
