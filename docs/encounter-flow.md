# Encounter Flow

This document describes the primary SpriteDex interaction loop.

---

# Primary Encounter Loop

1. User encounters organism
2. User takes or uploads photo
3. AI suggests candidate species
4. User selects or corrects species
5. GPS location is attached
6. Encounter metadata is added
7. Encounter is saved
8. Species collection updates
9. Species page reflects new observation

---

# Encounter Metadata

Potential metadata:
- date and time
- GPS coordinates
- weather
- habitat
- life stage
- abundance
- notes
- media attachments
- confidence level
